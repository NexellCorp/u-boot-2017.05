Nexell nxp3220 Timer

Required properties:

- compatible : should be "nexell,nxp3220-timer"
- reg : Specifies base physical address and size of the registers.
- clocks : Specifies clocks that drives the counter.
- clock-names : associated controller clock names.
- clock-frequency : The frequency of the clock that drives the counter, in Hz.

Example:

timer0: timer@208c0000 {
	compatible = "nexell,nxp3220-timer";
	reg = <0x208c0000 0x100>;
	clocks = <&clock_sys TIMER_0_TCLK0_CLK>;
	clock-names = "timer0";
	clock_frequency = <100000000>;
};
